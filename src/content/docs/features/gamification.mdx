HealthLog keeps users motivated with an achievement system. Progress and milestones are rewarded with badges and points.

## Achievement Categories

38 achievements spread across several categories:

### Total Intakes

Milestones for the total number of medication intakes (e.g., 10, 50, 100, 500, 1000).

### Streaks

Consecutive series without interruption:

| Streak Type | Description |
|------------|------------|
| Compliance Streak | Consecutive days with 100% adherence |
| Punctuality Streak | Consecutive intakes within the time window |
| Login Streak | Consecutive days with app usage |
| BMI in Target Range | Consecutive readings in the normal range |
| Blood Pressure in Target Range | Consecutive readings in the normal range |
| Heart Rate in Target Range | Consecutive readings in the normal range |

### Passkey Usage

Achievements for using passkeys to sign in (security incentive).

### Bug Reports

Rewards for reporting bugs through the built-in bug report system.

## Point System

Each achievement awards points. The total score is displayed in the user profile. Higher tiers of an achievement (e.g., 100 vs. 1000 intakes) award more points.

## Notifications

When an achievement is unlocked, the user receives a notification:

- In-app notification with achievement details
- Optionally via configured notification channels (Telegram, ntfy, Web Push)

## Progress Tracking

The profile shows current progress toward locked achievements:

- Percentage progress bar
- Next milestone
- Current streak length

:::note
The gamification system is designed purely as positive reinforcement. There are no penalties or negative consequences for broken streaks.
:::
