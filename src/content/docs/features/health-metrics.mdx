HealthLog tracks and analyzes key health metrics. All metrics can be entered manually or synced automatically via integrations (e.g., Withings), and are displayed as trend visualizations.

## Supported Metrics

| Metric | Unit | Automatic Analysis | Withings Sync |
|--------|------|-------------------|----|
| Weight | kg | BMI calculation, trend analysis | Yes |
| Blood Pressure (Sys/Dia) | mmHg | WHO/ESC/ESH classification | Yes |
| Heart Rate | bpm | Trend analysis | Yes |
| Body Fat Percentage | % | Trend analysis | Yes |
| Total Body Water | kg | Trend analysis | Yes (Body+ scales) |
| Bone Mass | kg | Trend analysis | Yes (Body+ scales) |
| Sleep Duration | hours | Trend analysis | — |
| Activity (Steps) | steps | Trend analysis | Yes |
| Blood Glucose | mg/dL or mmol/L | Per-context ranges (ADA 2024) | — |

Total body water and bone mass were added in 1.3.0. Both are stored canonically in kilograms — matching what Withings Body+ scales report (`hydration` measure type 77 and `bone_mass` measure type 88) and what Health Connect's `TotalBodyWaterRecord` / `BoneMassRecord` provide. Anyone with a Withings connection sees them flowing in automatically on the next sync, no extra config.

## Data Entry

### Single Entry

Individual readings can be logged directly from the dashboard. Each entry is automatically timestamped.

### Batch Entry

A batch mode is available for backfilling multiple readings at once — useful for transferring data from a paper diary or another tracking system.

## Trend Analysis

All metrics are analyzed across three time ranges:

- **7 days** -- Short-term trends
- **30 days** -- Monthly trends
- **90 days** -- Long-term development

Visualizations are rendered with Recharts (line and bar charts).

## BMI Calculation

BMI is calculated automatically from weight and height. Height is set once in the user profile.

```
BMI = weight / (height_m * height_m)
```

:::note
Height must be configured in your user profile for automatic BMI calculation to work.
:::

## Blood Pressure Classification

Readings are classified according to the 2018 ESC/ESH guidelines and WHO classification:

| Category | Systolic (mmHg) | Diastolic (mmHg) |
|----------|-----------------|-------------------|
| Optimal | < 120 | < 80 |
| Normal | 120--129 | 80--84 |
| High Normal | 130--139 | 85--89 |
| Grade 1 Hypertension | 140--159 | 90--99 |
| Grade 2 Hypertension | 160--179 | 100--109 |
| Grade 3 Hypertension | >= 180 | >= 110 |

:::tip
Classification is based on the higher value. If systolic and diastolic fall into different categories, the higher grade applies.
:::

## Charts and Visualizations

All charts are built with Recharts and include the following types:

- **Line charts** for weight, BMI, and heart rate
- **Dual-line charts** for blood pressure (Sys/Dia)
- **Bar charts** for steps and sleep duration
- Color coding by classification level for blood pressure

## Doctor's Report (PDF Export)

HealthLog generates a structured PDF report for doctor visits.

- **Format**: PDF via jsPDF (generated client-side)
- **Content**: Blood pressure history, weight trends, BMI, medication plan
- **Classification**: Based on 2018 ESC/ESH guidelines
- **Time range**: Freely selectable

:::caution
The PDF is generated entirely in the browser. For very large datasets, generation may take a few seconds.
:::
