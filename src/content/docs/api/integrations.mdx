## Withings

Integration with Withings devices (scales, blood pressure monitors) for automatic data import.

### Connect

```
GET /api/withings/connect
```

**Auth:** Session Cookie

Redirects to the Withings OAuth authorization flow.

### OAuth Callback

```
GET /api/withings/callback
```

Processes the OAuth callback from Withings and stores the credentials.

### Connection Status

```
GET /api/withings/status
```

**Auth:** Session Cookie

```json
{
  "data": {
    "connected": true,
    "lastSync": "2026-03-08T06:00:00.000Z",
    "deviceCount": 2
  },
  "error": null
}
```

### Sync

```
POST /api/withings/sync
```

**Auth:** Session Cookie

Triggers a manual sync of Withings data.

```bash
curl -s -X POST https://your-instance.example.com/api/withings/sync \
  -H "Cookie: healthlog_session=abc123"
```

### Disconnect

```
POST /api/withings/disconnect
```

**Auth:** Session Cookie

### Manage Credentials

```
GET /api/withings/credentials
PUT /api/withings/credentials
```

**Auth:** Session Cookie

Manages Withings API credentials (Client ID, Client Secret).

### Webhook

```
POST /api/withings/webhook
```

**Auth:** HMAC signature

Receives real-time notifications from Withings when new measurements are available. The request is verified via HMAC signature.

## MoodLog

Integration with the MoodLog app for importing mood data.

### Status

```
GET /api/integrations/moodlog/status
```

**Auth:** Session Cookie

```json
{
  "data": {
    "connected": true,
    "lastSync": "2026-03-08T19:30:00.000Z",
    "entriesImported": 47
  },
  "error": null
}
```

### Sync

```
POST /api/integrations/moodlog/sync
```

**Auth:** Session Cookie

Triggers a manual import of MoodLog data.

### Webhook

```
POST /api/integrations/moodlog/webhook
```

**Auth:** HMAC signature

Receives real-time notifications from MoodLog when new entries are created. The request is verified via HMAC signature.
