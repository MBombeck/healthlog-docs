## Notification Preferences

```
GET /api/notifications/preferences
PUT /api/notifications/preferences
```

**Auth:** Session Cookie

Manages channel preferences and per-event opt-out settings for each channel.

```bash
curl -s https://your-instance.example.com/api/notifications/preferences \
  -H "Cookie: healthlog-session=abc123"
```

**Response:**

```json
{
  "data": {
    "channels": {
      "webPush": true,
      "telegram": true,
      "ntfy": false
    },
    "events": {
      "MEDICATION_REMINDER": { "webPush": true, "telegram": true, "ntfy": false },
      "INSIGHT_READY": { "webPush": true, "telegram": false, "ntfy": false }
    }
  },
  "error": null
}
```

## Web Push

### Subscribe

```
POST /api/notifications/web-push
```

**Auth:** Session Cookie

Registers a Web Push subscription.

```json
{
  "endpoint": "https://fcm.googleapis.com/fcm/send/...",
  "keys": {
    "p256dh": "...",
    "auth": "..."
  }
}
```

### Unsubscribe

```
DELETE /api/notifications/web-push
```

### VAPID Key

```
GET /api/notifications/vapid
```

**Auth:** None

Returns the public VAPID key required for Web Push subscriptions.

```json
{
  "data": {
    "publicKey": "BNx..."
  },
  "error": null
}
```

## Telegram

### Configuration

```
GET /api/settings/telegram
PUT /api/settings/telegram
```

**Auth:** Session Cookie

```json
{
  "botToken": "123456:ABC-...",
  "chatId": "987654321",
  "webhookEnabled": true
}
```

### Test Message

```
POST /api/settings/telegram/test
```

**Auth:** Session Cookie

Sends a test message through the configured Telegram bot.

```bash
curl -s -X POST https://your-instance.example.com/api/settings/telegram/test \
  -H "Cookie: healthlog-session=abc123"
```
