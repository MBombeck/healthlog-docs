## Measurement Types

| Type | Description | Unit |
|------|-------------|------|
| `WEIGHT` | Body weight | kg |
| `BLOOD_PRESSURE_SYS` | Systolic blood pressure | mmHg |
| `BLOOD_PRESSURE_DIA` | Diastolic blood pressure | mmHg |
| `PULSE` | Pulse / heart rate | bpm |
| `BODY_FAT` | Body fat percentage | % |
| `SLEEP_DURATION` | Sleep duration | hours |
| `ACTIVITY_STEPS` | Steps | steps |

## List Measurements

```
GET /api/measurements
```

**Auth:** Session Cookie or Bearer Token

**Query Parameters:**

| Parameter | Type | Description |
|-----------|------|-------------|
| `type` | string | Filter by measurement type |
| `from` | ISO 8601 | Start date |
| `to` | ISO 8601 | End date |
| `limit` | number | Max results (default: 20) |
| `offset` | number | Offset for pagination |
| `sortBy` | string | Sort field (e.g. `measuredAt`) |
| `sortDir` | `asc` / `desc` | Sort direction |

```bash
curl -s "https://your-instance.example.com/api/measurements?type=WEIGHT&limit=5&sortDir=desc" \
  -H "Cookie: healthlog_session=abc123"
```

**Response:**

```json
{
  "data": [
    {
      "id": "clx...",
      "type": "WEIGHT",
      "value": 81.5,
      "measuredAt": "2026-03-08T07:30:00.000Z",
      "notes": null,
      "createdAt": "2026-03-08T07:31:12.000Z"
    }
  ],
  "error": null,
  "meta": { "total": 142, "limit": 5, "offset": 0 }
}
```

## Create Measurement

```
POST /api/measurements
```

**Auth:** Session Cookie or Bearer Token

Supports both single values and batch creation (array).

### Single Measurement

```json
{
  "type": "WEIGHT",
  "value": 81.5,
  "measuredAt": "2026-03-08T07:30:00.000Z",
  "notes": "Measured fasting"
}
```

### Blood Pressure (Batch)

A blood pressure reading consists of two related measurements:

```bash
curl -s https://your-instance.example.com/api/measurements \
  -H "Content-Type: application/json" \
  -H "Cookie: healthlog_session=abc123" \
  -d '[
    {
      "type": "BLOOD_PRESSURE_SYS",
      "value": 125,
      "measuredAt": "2026-03-08T08:00:00.000Z"
    },
    {
      "type": "BLOOD_PRESSURE_DIA",
      "value": 82,
      "measuredAt": "2026-03-08T08:00:00.000Z"
    },
    {
      "type": "PULSE",
      "value": 68,
      "measuredAt": "2026-03-08T08:00:00.000Z"
    }
  ]'
```

**Response** `201`:

```json
{
  "data": [
    { "id": "clx1...", "type": "BLOOD_PRESSURE_SYS", "value": 125, "measuredAt": "2026-03-08T08:00:00.000Z" },
    { "id": "clx2...", "type": "BLOOD_PRESSURE_DIA", "value": 82, "measuredAt": "2026-03-08T08:00:00.000Z" },
    { "id": "clx3...", "type": "PULSE", "value": 68, "measuredAt": "2026-03-08T08:00:00.000Z" }
  ],
  "error": null
}
```

## Single Measurement

### Retrieve

```
GET /api/measurements/[id]
```

### Update

```
PUT /api/measurements/[id]
```

```json
{
  "value": 82.0,
  "notes": "Corrected"
}
```

### Delete

```
DELETE /api/measurements/[id]
```

**Auth:** Session Cookie or Bearer Token. Users can only modify their own measurements.
