## Mood Values

| Value | Label (German) | English | Score |
|-------|---------------|---------|-------|
| `SUPER_GUT` | Super gut | Excellent | 5 |
| `GUT` | Gut | Good | 4 |
| `OKAY` | Okay | Okay | 3 |
| `SCHLECHT` | Schlecht | Bad | 2 |
| `LAUSIG` | Lausig | Terrible | 1 |

:::note
Mood enum values use German labels as the app's primary UI language is German. The `score` field (1–5) is language-independent.
:::

## List Entries

```
GET /api/mood-entries
```

**Auth:** Session Cookie or Bearer Token

**Query Parameters:**

| Parameter | Type | Description |
|-----------|------|-------------|
| `mood` | string | Filter by mood value |
| `from` | ISO 8601 | Start date |
| `to` | ISO 8601 | End date |

```bash
curl -s "https://your-instance.example.com/api/mood-entries?from=2026-03-01&to=2026-03-08" \
  -H "Cookie: healthlog_session=abc123"
```

**Response:**

```json
{
  "data": [
    {
      "id": "clx...",
      "mood": "GUT",
      "score": 4,
      "tags": ["Exercise", "Slept well"],
      "notes": "Productive day",
      "createdAt": "2026-03-08T20:00:00.000Z"
    }
  ],
  "error": null
}
```

## Create Entry

```
POST /api/mood-entries
```

**Auth:** Session Cookie

```bash
curl -s https://your-instance.example.com/api/mood-entries \
  -H "Content-Type: application/json" \
  -H "Cookie: healthlog_session=abc123" \
  -d '{
    "mood": "GUT",
    "score": 4,
    "tags": ["Exercise", "Slept well"],
    "notes": "Productive day"
  }'
```

**Response** `201`:

```json
{
  "data": {
    "id": "clx...",
    "mood": "GUT",
    "score": 4,
    "tags": ["Exercise", "Slept well"],
    "notes": "Productive day",
    "createdAt": "2026-03-08T20:00:00.000Z"
  },
  "error": null
}
```

## Single Entry

```
GET    /api/mood-entries/[id]     — Retrieve details
PUT    /api/mood-entries/[id]     — Update
DELETE /api/mood-entries/[id]     — Delete
```

**Auth:** Session Cookie. Users can only manage their own entries.

## Mood Analytics

```
GET /api/mood/analytics
```

**Auth:** Session Cookie

Returns aggregated mood data including averages, distribution, and trends.

```json
{
  "data": {
    "averageScore": 3.6,
    "distribution": {
      "SUPER_GUT": 5,
      "GUT": 12,
      "OKAY": 8,
      "SCHLECHT": 3,
      "LAUSIG": 1
    },
    "trend": "STABLE"
  },
  "error": null
}
```
