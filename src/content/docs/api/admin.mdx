All endpoints on this page require the **Admin** role.

## System Status

```
GET /api/admin/status
```

**Auth:** Session Cookie (Admin)

Returns a system overview including user count, measurements, worker status, integration status, and version.

```bash
curl -s https://your-instance.example.com/api/admin/status \
  -H "Cookie: healthlog_session=abc123"
```

**Response:**

```json
{
  "data": {
    "version": "1.4.2",
    "users": 3,
    "measurements": 1842,
    "medications": 7,
    "moodEntries": 156,
    "worker": { "status": "RUNNING", "pendingJobs": 0 },
    "integrations": {
      "withings": { "connected": true },
      "telegram": { "configured": true },
      "glitchtip": { "configured": true }
    }
  },
  "error": null
}
```

## User Management

### List All Users

```
GET /api/admin/users
```

### Single User

```
GET /api/admin/users/[id]
```

### Delete User

```
DELETE /api/admin/users/[id]
```

### Reset Password

```
POST /api/admin/users/[id]/reset-password
```

Generates a temporary password for the user.

```bash
curl -s -X POST https://your-instance.example.com/api/admin/users/clx.../reset-password \
  -H "Cookie: healthlog_session=abc123"
```

**Response:**

```json
{
  "data": {
    "temporaryPassword": "xK9m!pQ2..."
  },
  "error": null
}
```

## Audit Log

```
GET /api/admin/audit-log
```

**Auth:** Session Cookie (Admin)

## Database Statistics

```
GET /api/admin/data
```

**Auth:** Session Cookie (Admin)

Returns database usage statistics (table sizes, entries per type).

---

## Test Notifications

```
POST /api/admin/notifications/test
```

**Auth:** Session Cookie (Admin)

Sends a test notification through all configured channels.

### Reminder Check

```
POST /api/admin/notifications/reminder-check
```

**Auth:** Session Cookie (Admin)

Triggers a manual check for due medication reminders.

---

## Test Monitoring

### GlitchTip

```
POST /api/admin/monitoring/glitchtip-test
```

**Auth:** Session Cookie (Admin)

Sends a test event to GlitchTip (Sentry-compatible).

### Umami

```
POST /api/admin/monitoring/umami-test
```

**Auth:** Session Cookie (Admin)

Sends a test event to Umami Analytics.
